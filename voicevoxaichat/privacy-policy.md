# プライバシーポリシー

最終更新日: 2026-05-07

「VoiceVox AI Chat」(以下「本アプリ」) における個人情報の取扱いについて、以下のとおり定めます。

## 1. 取得する情報

### 1.1 ユーザー情報 (Sign in with Apple 経由)

- ユーザー識別子 (Apple が発行する一意の文字列)
- 氏名 (ユーザーが共有を選択した場合)
- メールアドレス (ユーザーが共有を選択した場合、または匿名中継アドレス)

### 1.2 課金情報

- App Store transaction ID
- 購入商品 ID、購入日時、購入金額
- 返金状態

(クレジットカード情報・銀行口座情報等の決済手段情報は Apple が処理するため、本アプリは取得しません)

### 1.3 利用情報

- AI チャットの入力テキスト・出力テキスト (一時的な処理に必要な範囲)
- 使用モデル、トークン消費量
- リクエスト日時

### 1.4 技術情報

- IP アドレス (Cloudflare Workers 経由で接続元の IP を一時的に取得)
- アプリケーションログ (エラーログ・パフォーマンスログ)

## 2. 利用目的

- 本アプリのサービス提供 (AI チャット応答の生成、ポイント残高管理)
- 課金処理および返金対応
- 不正利用の検知および防止
- サービスの品質向上

## 3. 第三者への提供 (AI 提供事業者)

本アプリは以下の AI 提供事業者のサービスをバックエンドに利用しており、ユーザーが選択したモデルにより入力データは該当する事業者に送信されます。

| 事業者 | サービス | データ処理地域 |
|---|---|---|
| Anthropic, PBC | Claude API | アメリカ合衆国 |
| Alibaba Cloud (DashScope International) | Qwen API | シンガポール (場合により中国本土に移管される可能性あり) |

各事業者の利用規約・プライバシーポリシー:

- Anthropic: <https://www.anthropic.com/legal/privacy>
- Alibaba Cloud: <https://www.alibabacloud.com/help/en/legal/privacy-policy>

ユーザーは AI モデル選択により送信先を選択できます。

## 4. インフラストラクチャ事業者

本アプリの運用に以下のインフラストラクチャ事業者を利用しています。

| 事業者 | 用途 | データ処理地域 |
|---|---|---|
| Apple Inc. | Sign in with Apple、App Store、StoreKit | 米国・グローバル |
| Cloudflare, Inc. | API プロキシ、CDN | グローバル (主に Tokyo Edge) |
| Fly.io | アプリケーションサーバ | 東京 (NRT region) |
| Neon | PostgreSQL データベース | アジア region |

## 5. 保管期間

- ユーザーアカウント情報: アカウント存続中
- 課金情報 (payments / refunds): **法定保存期間 (7年)** に従い、ユーザー削除後も user_id を匿名化した上で保管
- 利用情報: 30日間 (運用上必要な範囲)
- 技術情報 (ログ): 90日間

## 6. ユーザーの権利

ユーザーは以下の権利を行使できます。

- **アカウント削除**: 本アプリ内「設定 > アカウント管理」から即時削除 または 30日後の削除予約
- **個人情報の開示・訂正・利用停止**: お問い合わせ窓口より連絡

アカウント削除時、課金情報を除くデータは速やかに削除されます。課金情報は法定保存期間中、user_id を匿名化した形で保管されます。

## 7. 改定

本ポリシーは予告なく改定される場合があります。重要な変更があった場合は、本アプリ内またはホームページで通知します。
